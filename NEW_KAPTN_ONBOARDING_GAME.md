# üé® UI/UX Design Specifications

## Visual Language
- **Color Palette:** Black background, white text, accent colors only for protocol indicators
- **Typography:** Monospace for system messages, clean sans-serif for content
- **Animation:** Minimal, purposeful. Text appears with slight delay for gravitas
- **Sound:** Optional subtle UI sounds (radar ping, system confirm)

## Interaction Principles
- No timers (except narrative ones)
- No progress bars that create pressure  
- Each choice requires deliberate click
- Cannot go back (emphasizes commitment)
- No "correct" visual feedback (no green checks or red X's)

## Accessibility
- High contrast mode available
- Keyboard navigation supported
- Screen reader compatible
- Multiple language support

## Profile Generation Logic
- Map response patterns to decision style descriptors
- No scoring system (avoid gamification)
- Generate unique insight based on combination
- Store as initial OAS (Ontological Alignment Score) baseline

## Integration Points
- Profile feeds into CFR (Counterfactual Reasoning) system
- Influences initial alternative suggestion ranking
- Establishes baseline for "thinks like me" calibration
- Updates over time based on actual usage patterns

# üé™ Why This Works

## 1. **Ceremonial Without Being Performative**
- The oath creates genuine commitment moment
- The visual progression feels significant
- No artificial gamification

## 2. **Informative Without Being Evaluative**  
- Reveals decision patterns without judgment
- User learns about themselves genuinely
- No "good" or "bad" profiles

## 3. **Bridge-Themed Throughout**
- Every scenario uses navigation/ship metaphors
- Consistent with KAPTN brand language  
- Feels like qualification, not test

## 4. **Practical Foundation**
- Directly maps to KAPTN protocols (K-A-T-P-N)
- Generates actionable data for system
- Sets up trust relationship

# üìù Copywriting Principles

## Voice Guidelines
- **Precise, not flowery** - "Unknown signal detected" not "A mysterious opportunity appears"
- **Respectful, not playful** - User is professional, not playing a game
- **Systemic, not personal** - "The bridge calibrates" not "I'm learning about you"
- **Mission-oriented** - Always forward-looking, never nostalgic

## Language to AVOID
- "Personality type"
- "You are a [label]"  
- "Score" or "rating"
- "Most people choose..."
- "Correct answer"

## Language to USE
- "Protocol"
- "Pattern"
- "Tendency"  
- "Navigation style"
- "Bridge configuration"
